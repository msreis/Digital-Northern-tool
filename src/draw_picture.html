Digital Northern Picture Generator - Beta Version (v0.7)

This tool generates a Digital Northern picture. Please input the requested data exactly as described below.

Select the color for the picture: Red Green Blue Whatever

Input the relative abundance and the text for the picture, one gene per line.
Here there are some input data samples: Example 1; Example 2; Example 3; Example 4.



OBS: Do not use brackets "[" and "]" within the product text. Make sure that you input the
correct relative abundance values, and also placed the brackets between the text.
